# RYÛKO 

Currently v0.1-devbuild

Download latest version for Windows-x64 or OSX-Silicone from https://github.com/evilgoosey/RYUKOpublic/releases 

Small demo of my planned martial arts fighting card game that currently just runs in the terminal window, based on the RYÛKO card game, and featuring dynamic card generation and strategic range management.

Testing is currently:
    -By invite only

## How to Play
1. Download the latest release from the Releases tab.
2. Extract the `.zip` file.
3. Run `CombatSim.exe`.

### Controls
- **A/Z**: Shift Range / Circle
- **0-8**: Play Card

## Reporting Bugs
Found a bug? Please open an https://github.com/evilgoosey/RYUKOpublic/issues on GitHub. 
Include:
1. What you were doing (e.g., "Tiger took turn at Striking range").
2. What happened (e.g., "HP went negative but Tiger didn't die").
3. The Round and turn number of the artifact.
4. game_log.txt readout from as close to that moment as possible.

## Known Issues
- NPC isn't able to use all cards/mechanics yet but good enough for fun games (techniques and discarding injuries).
- Occasional UI flicker during round transitions.
